## The ANC Paradox
Nigerian women attend checkups faithfully — then go home
to deliver alone. This structural dropout between antenatal
care and facility delivery is the hidden driver of Nigeria's
maternal mortality crisis.

This oracle quantifies that gap and predicts risk in real time.
